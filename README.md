# Multi-Step Form

A front-end multi-step form where users move through multiple steps, validate inputs, review details, and save progress locally.

## Live Demo

https://jfoster294.github.io/multi-step-form/

## Features

- Multi-step form layout
- Step progress indicator
- Step 1: Personal info
- Step 2: Developer goals
- Step 3: Project preferences
- Step 4: Review and submit
- Next and back navigation
- Input validation
- Email validation
- Saved draft with local storage
- Saved current step
- Clear saved form
- Confirmation message
- Responsive design for mobile and desktop

## Built With

- HTML
- CSS
- JavaScript
- Forms
- Validation
- Local Storage

## Project Goal

This project was built as part of my front-end developer portfolio to practice form validation, multi-step UI logic, state management, local storage, accessibility-friendly structure, and responsive design.
